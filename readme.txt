== Description ==

This is the extension pack for avalon23. Attention this is not a standalone plugin. This is an additional plugin for [Avalon23 Products Filter for WooCommerce](https://wordpress.org/plugins/avalon23-products-filter-for-woocommerce/)


Latest PHP 8.x - COMPATIBLE!

Docs: [Extension pack for Avalon23](https://avalon23.dev/document/avalon23-extension-pack/)

Demo site : [demo.avalon23.dev](https://demo.avalon23.dev)

### Extension pack Avalon23 Features:

&#9989;&nbsp; **Auto price** Automatic determination of the minimum and maximum price levels on custom pages

&#9989;&nbsp; **Preloader** You can configure the preloader (background color and main image) when ajax filtering works

== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Set product taxonomies in the plugin settings tab of the WooCommerce settings page
* Drop the Products Filter widget in the sidebar or shortcode in  content.
* Use.


== Frequently Asked Questions ==

Q: Where are the settings for this plugin?
R: All settings will automatically appear on Avalon23 Products Filter for WooCommerce plugin page
